# thinkBridge

##### All platforms
This project was build with node : v16.8.0, yarn : v1.22.11

##### Android
- Java JDK
- Android SDK

##### iOS
- Cocoapods
- Xcode 13.0.0
- Xcode CLI tools
- React Native CLI


### Cleaning the project
```shell
yarn clean
```

### Install dependencies
```shell
yarn
```
### Install pods
```shell
yarn pods
```

### Run iOS app
```shell
yarn ios
```
### Run Android app
```shell
yarn android
```
